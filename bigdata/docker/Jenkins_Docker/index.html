启动一个Container

  1. 创建本地Jenkins数据目录，将该目录映射到docker中的Jenkins目录，方便备份数据
  2. 8080端口是Jenkins web service的默认端口，将宿主机的端口映射到8080端口，可以直接访问
  3. –name 命名为prod_jenkins2
  4. -d 后台运行
  5. -p 端口映射
  6. -v 文件目录映射
