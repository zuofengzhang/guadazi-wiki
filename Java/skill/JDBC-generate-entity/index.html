Entity

Entity或者DTO是Java中与数据表相对应的实体，将对象的属性封装有利于数据操作(如从数据库中读取某个实体，对这个实体操作，修改，新增等)，根据数据表生成Entity是必须应对的问题（当然可以选择手动敲），目前常用的方法有：

  • Hibernate工具: Eclipse和Intellij IDEA有相应的插件
